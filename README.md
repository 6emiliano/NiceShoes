# NiceShoes - Tienda de Zapatillas

## Descripcion

Simulador de tienda online de zapatillas con carrito de compras, sistema de stock y panel de administracion.

## Como ejecutar

Abrir el archivo index.html en el navegador o el link de github

Si hay problemas con el carrito o los datos guardados, probar abrir el link en modo incognito.

## Funcionalidades

- Catalogo de productos cargado desde JSON y API externa
- Carrito de compras con persistencia en LocalStorage
- Control de stock automatico
- Calculo de IVA y descuentos
- Panel de administracion para crear, buscar y filtrar productos
- Historial de compras
